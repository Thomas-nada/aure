## ğŸ§  Consensus Overview

A block is accepted if and only if:

1. Height increments correctly
2. `prevHash` matches the chain tip
3. Hash matches the exact serialized block
4. Hash satisfies the target
5. Exactly one coinbase transaction exists
6. Coinbase is the first transaction
7. Block reward matches consensus rules

Invalid blocks are rejected immediately.
